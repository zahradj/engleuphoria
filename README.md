# EnglEuphoria

EnglEuphoria is an English learning platform for kids, teens, and adults — combining live classrooms, AI tutoring, gamification, and a structured CEFR-aligned curriculum.

## Local development

Requirements: Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui
- Supabase
